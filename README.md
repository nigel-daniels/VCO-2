# VCO 2

This VCO based on the work of  LMNC [performance VCO](https://www.lookmumnocomputer.com/1222-performance-vco). I've swapped out the built in tuner for a sine wave generator, expanded the octave range by one and enabled hard/soft sync selection. Why, well I'm unlikely to go performing and I was thinking I'll just build a sigle tuning module to take advantage of the exposed trim pots. 

There are a couple of folders included, these contain footprints I used for the switches, import these as project specific items.

The PCB design is my own and I designed it as a two layer pcb made of two double sided pcbs.

I hope you find this useful, and a huge thanks to the LMNC site for some inspiration to do this version of a VCO, also to the lovely people at OnChip Systems (nee Curtis Electronics) who still manufacture and supply the CEM3340. I tracked them down here in San Jose to buy from the source ;)
